# Dashboard Screenshots

## Overview
This folder contains screenshots of the final Power BI dashboard developed as part of the **Business Sales Analytics Project**.

The screenshots provide a visual overview of the dashboard layout, key performance indicators (KPIs), interactive filters, and business insights generated from the Superstore dataset.

---

## Dashboard Highlights

### Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Total Orders
- Total Customers

### Sales Performance Analysis
- Revenue Trend Over Time
- Top 10 Products by Sales
- Sales by Category
- Sales by Region

### Profitability Analysis
- Profit by Category
- Profit by Region

### Interactive Filtering
- Region Filter
- Year Filter
- Category Filter

---

## Purpose
The screenshots allow users, recruiters, and reviewers to quickly understand the dashboard design and business insights without opening the Power BI file.
