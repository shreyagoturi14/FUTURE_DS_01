# Power BI Dashboard

## Project Overview

This Power BI dashboard was developed using the Superstore Dataset to analyze sales performance, profitability, customer trends, product performance, and regional insights.
The dashboard converts raw business data into interactive visualizations that support data-driven decision-making.

---

## Dashboard Preview
The dashboard includes:
- Total Sales Analysis
- Total Profit Analysis
- Total Orders Analysis
- Total Customers Analysis
- Revenue Trend Over Time
- Top 10 Products by Sales
- Sales by Category
- Profit by Category
- Sales by Region
- Profit by Region

---

## Interactive Filters

Users can dynamically filter dashboard insights using:
- Region
- Year
- Category

- Technology generated the highest profit.
- West region achieved the highest sales performance.
- West region generated the highest profit.
- Revenue showed strong growth over time.
- A small group of products contributed significantly to overall sales.
